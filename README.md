**ポイント**
- 親スケールは常に **C メロディックマイナー**（音集合は同じ）。ルートが変わるのはモードの開始位置だけです。
- ルート音は ♭表記で統一（例: C# ではなく Db）。
- ジャズ/フュージョンでは特に **オルタード**（altドミナント）と **メロディックマイナー**（マイナーi）が頻出。
- 指板上同じポジションで7つのモードが使えるため、1つのスケールポジションを覚えるだけで多様な響きが出せます。
